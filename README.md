# Personal Portfolio - Diwakar Mahato Sudi

Welcome to the repository for my personal portfolio website! This project showcases my skills, experience, and projects as a Computer Science student. You can view the live site at [diwakarmahatosudi.com](https://diwakarmahatosudi.com).

## Overview

This portfolio is a static web application designed to be clean, responsive, and engaging. It highlights my background in software development, web technologies, and AI.

### Key Sections:
- **About Me:** Information about my education, skills, and background.
- **Experience:** Details about my internships and research roles.
- **Projects:** A showcase of my recent work, including an AI Image Generator, Amazon Clone, and more.
- **Certifications:** Links to certifications I've earned.
- **Contact:** A form to get in touch with me directly.

## Technologies Used

*   **HTML5:** For the structure and content of the pages.
*   **CSS3:** For styling, layout (Flexbox/Grid), and animations.
*   **JavaScript:** For interactivity, DOM manipulation, and dynamic features.
*   **Particles.js:** Used for the interactive background effect in the hero section.
*   **AOS (Animate On Scroll):** Used to trigger CSS animations as elements scroll into view.

## Features

*   **Responsive Design:** Adapts smoothly to different screen sizes (mobile, tablet, desktop).
*   **Interactive UI:** Includes hover effects, typing animations, and scroll-triggered reveals.
*   **Modern Aesthetics:** Utilizes a custom color palette, modern typography, and clean layout principles.
*   **Contact Form:** Functional contact form (powered by Formspree).

## Running Locally

To run this project on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Diwakarmahatosudi/portfolio.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd portfolio
    ```
3.  **Open `index.html`:**
    You can simply double-click the `index.html` file to open it in your browser. For a better development experience (to handle potential CORS issues with local files), use a local server:
    *   **VS Code:** Install the "Live Server" extension and click "Go Live".
    *   **Node.js:** Run `npx http-server` in the terminal and open the provided `localhost` URL.
    *   **Python:** Run `python3 -m http.server` and open `http://localhost:8000`.
